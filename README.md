## Project Overview
This project involves a comprehensive analysis of **4 years of sales data** (9,800+ orders across 50 U.S. states) to evaluate profitability and customer performance.

## Key Insights 💡
* **Profitability Drivers:** Identified **Technology** as the top-performing category, contributing to **~38% of total profit** and showing **45% growth**.
* **Regional Analysis:** Mapped sales across 50 states to identify high-volume vs. high-margin territories.
* **Seasonal Trends:** Uncovered a **2.5x sales spike in Q4**, indicating strong year-end revenue concentration and seasonal demand.
* **Customer Segmentation:** Analyzed top 5 profit-making customers to understand high-value retention patterns.

## Tools & Techniques 🛠️
* **Excel:** Advanced Pivot Tables, VLOOKUP/XLOOKUP, and Data Formatting.
* **Visualization:** Created a dynamic dashboard using Bar Charts, Area Charts, Pie Charts, and Interactive Slicers for Category and Year.
* **Data Modeling:** Structured raw transaction data into a clean, report-ready format.
